#  KitaabBazaar — Used Book Marketplace System

##  Project Description
KitaabBazaar is a Java-based desktop marketplace application developed to help students and users buy and sell used books online. The system provides separate dashboards for buyers, sellers, and admins, allowing users to manage books, orders, and accounts efficiently. The project integrates Java Swing for the graphical user interface and MySQL with JDBC for database connectivity while implementing core Object-Oriented Programming (OOP) concepts such as inheritance, polymorphism, abstraction, encapsulation, interfaces, and exception handling.

##  Purpose & Main Modules

The purpose of this project is to apply Object-Oriented Programming (OOP) principles and Database Management concepts to a real-world online marketplace system.

### Main Modules:

* **Admin Dashboard:** Manage users, books, and marketplace activities.
* **Buyer Module:** Browse books and place orders.
* **Seller Module:** Add, update, and manage book listings.
* **Database Connectivity:** JDBC integration with MySQL database.
* **User Interface:** GUI built using Java Swing.
* **Exception Handling:** Custom exceptions for input validation and system errors.

##  How to Run

Follow these steps to get the project running locally:

1. **Prerequisites**
   - Install Java JDK 11 or above
   - Install MySQL Server
   - Install any Java IDE (IntelliJ, NetBeans, or Eclipse)

2. **Database Setup**
   - Open MySQL Workbench
   - Import the provided `.sql` file from the project directory

3. **IDE Setup**
   - Open the project in your IDE
   - Add MySQL Connector JAR from the `/lib` folder to project libraries

4. **Configure Database**
   - Open `DBConnection.java`
   - Update your MySQL username and password

5. **Execution**
   - Run the `LoginScreen.java` file from the `src/ui` package
